# Experiment 3.1.1 – Login Form with React State Management

## Aim
To create a login form using React state management for client-side validation and simulated authentication.

## Setup

```bash
cd login-form
npm install
npm start
```

App runs at http://localhost:3000

## Note
This experiment demonstrates client-side validation and a simulated authentication flow (no backend authentication service).
