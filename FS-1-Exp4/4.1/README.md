# Experiment 4.1 - CLI Employee Management System

## Aim
To build a command-line employee management system using Node.js with array-based storage.

## Objectives
1. Implement CRUD operations for employees
2. Create interactive CLI menus
3. Validate user input
4. Handle array operations

## Hardware/Software Requirements
- Node.js 18+
- Terminal/Command Prompt
- No additional dependencies

## How to Run

```bash
node employee-management.js
```

## Features
- **Add Employee** – Enter name, position, and salary
- **List Employees** – View all employee records with ID, name, position, salary
- **Update Employee** – Update employee details by ID
- **Delete Employee** – Remove an employee by ID
- **Input Validation** – Validates name, position, and salary fields
